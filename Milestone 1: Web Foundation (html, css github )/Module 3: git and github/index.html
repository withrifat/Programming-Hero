Git and GitHub

Git is a version control system that allows you to track changes in your code and collaborate with others. GitHub is a platform that hosts Git repositories, making it easier to share and collaborate on projects.

Key Concepts

  • Repository: A storage space for your project, which can be local or hosted on GitHub.
  • Commit: A snapshot of your project at a specific point in time.
  • Branch: A parallel version of your repository where you can make changes without affecting the main codebase.
  • Merge: Combining changes from one branch into another.

Basic Commands

    git init
    git clone 
    git add 
    git commit -m "commit message"
    git push
    git pull
    git branch
    git checkout 
    git merge

Practice

To get hands-on experience, create a new repository on GitHub, clone it to your local machine, make some changes, commit them, and push them back to GitHub.

Remember to practice branching and merging to understand how to manage different versions of your project effectively.
